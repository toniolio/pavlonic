# Privacy Notes

This repo is a public portfolio build of the Pavlonic platform.

## What is intentionally NOT included
- Private specs and source materials: `docs-private/` (gitignored)
- Any real extracted study data / databases: `data/private/` (gitignored)
- API keys and secrets: `.env*` (gitignored)

## What IS included
- Application code and infrastructure (without secrets)
- Demo/synthetic data and fixtures when needed (`data/demo/`)
- Public-safe documentation about architecture and approach
